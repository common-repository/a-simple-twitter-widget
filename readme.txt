== Description == 
Very simple twitter widget. Made with TwitterOAuth from [Abraham Williams](http://abrah.am "Abraham Williams").
CSS made with SASS/Compass, very simple and easy to custom.

== Installation ==
Just download, install and activate from the wordpress extension section.
